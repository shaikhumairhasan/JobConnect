JOB CONNECT

"Job Connect" is an online job portal web application created as part of a college project. This web application is developed using Java, Spring boot, HTML, CSS, Thymeleaf, and MySQL. The aim of this project is to make the process of job search simple and easy. It has a user friendly interface and offers a smooth experience. This project is developed by:
